# Historical Analysis Graphs

This directory preserves a static set of graph pages from the original Pebble
project work.

These files are historical outputs only. They are included as reference
material from the original project and do not define the Pebble architecture,
runtime requirements, or expected deployment structure.

Only the rendered graph site is kept here:

- `index.html`
- per-platform graph pages such as `fred/`, `goob/`, `minilamp/`, and
  `bumperbot/`
- the preserved `router_impairment/` comparison page

The underlying analysis scripts, raw datasets, generated reports, and
experiment-specific tooling are intentionally not included in this repository.

Some preserved graph text may still reference an earlier project name or the
original platform names. That wording is historical and is left intact inside
the static graph assets.
